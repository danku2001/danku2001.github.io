Overview

2025

A full-stack web application for managing campus areas and sensors. The system supports CRUD operations, data display, and a relational database-backed backend.

What I Built

  • Backend services in Node.js with REST endpoints for managing entities and data.
  • React UI with dynamic components for viewing and managing sensors and areas.
  • MySQL schema design for structured, queryable data storage.
